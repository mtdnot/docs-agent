# DiscoverWrapper

このページは、MCPサーバーの一覧を表示し、ユーザーが新しいサーバーを追加できるようにするためのものです。ページの上部には、現在の位置を示すパンくずリストがあり、ユーザーがどこにいるのかをすぐに理解できます。

サーバーの追加方法は2つあります。一つは「レジストリ」タブから、もう一つは「マニュアル」タブからです。レジストリタブでは、既に登録されているサーバーの一覧が表示されます。この一覧は、「認証済み」タブと「コミュニティ」タブの2つに分けられています。認証済みタブには、MCPチームによって確認されたサーバーが、コミュニティタブにはユーザーが提供したサーバーが表示されます。

それぞれのタブには、サーバーを検索するための検索ボックスがあります。検索ボックスに文字を入力すると、その文字を含むサーバーのみがリストに表示されます。また、各サーバーの右側には「インポート」ボタンがあり、これをクリックするとそのサーバーが自分のリストに追加されます。

マニュアルタブでは、サーバーの詳細情報を手動で入力して新しいサーバーを追加することができます。これには、サーバーの名前、説明、コマンド、環境変数などの情報が必要です。

この機能を使用すれば、ユーザーは自分のニーズに合ったサーバーを簡単に見つけて追加することができます。
